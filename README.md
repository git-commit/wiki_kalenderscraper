wiki_kalenderscraper
====================

Kalenderscraper fuer wiki.muc.ccc.de/kalender, Export zu .ics und .json

Exportiert alle Events der Tabelle zu
wiki_kalender.ics

Exportiert das naechste Event zu
nextevent.json
(fuer anzeiger)

python dependencies:
* beautifulsoup4
* icalendar
* requests
* html5lib
